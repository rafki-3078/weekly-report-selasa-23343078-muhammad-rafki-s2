# Job Sheet 5
  Jobsheet 05 ini membahas tentang struktur data Double Linked List dan Circular Linked List, yang merupakan dua jenis struktur data yang sering digunakan dalam pemrograman. Double Linked List adalah struktur data yang menghubungkan elemen-elemen dengan dua pointer dalam satu node, yaitu pointer next dan pointer prev, yang memungkinkan traversal baik ke depan maupun ke belakang. 
  
  Sementara itu, Circular Linked List adalah struktur data yang memberikan fleksibilitas dalam melewatkan elemen, dan dapat berupa single linked list atau double linked list, dengan pointer next dari elemen terakhir menunjuk ke elemen pertama.

# Job Sheet 11
  Jobsheet 11 ini membahas tentang dua metode pencarian data, yaitu Linear Search dan Binary Search. Linear Search adalah metode pencarian data yang dilakukan dengan cara membandingkan data yang dicari dengan setiap elemen dalam array secara sekuensial, mulai dari indeks pertama hingga ditemukan atau sampai dengan indeks terakhir. 
  
  Sedangkan Binary Search adalah metode pencarian data yang dilakukan dengan cara membandingkan data yang dicari dengan elemen tengah dari array yang sudah diurutkan, jika data yang dicari lebih besar dari elemen tengah maka pencarian dilanjutkan ke bagian kanan array, jika lebih kecil maka pencarian dilanjutkan ke bagian kiri array. 
  
  Jobsheet ini juga menyediakan beberapa contoh kode program untuk mengimplementasikan kedua metode tersebut dalam bahasa pemrograman C. Selain itu, jobsheet ini juga memberikan latihan dan tugas untuk memperdalam pemahaman tentang metode pencarian data.

# Job Sheet 12
  Jobsheet 12 ini membahas tentang konsep penjelajahan data Tree Traversal dan pencarian Binary Tree dalam pemrograman C. Tree Traversal adalah metode untuk mengunjungi dan mendata setiap simpul/node dalam pohon data. Ada tiga metode utama untuk melakukan Tree Traversal, yaitu Inorder Traversal, Preorder Traversal, dan Postorder Traversal. Inorder Traversal mengunjungi semua simpul di sebelah kiri subtree, kemudian mengunjungi root node, dan akhirnya mengunjungi semua simpul di sebelah kanan subtree. 
  
  Preorder Traversal mengunjungi root node terlebih dahulu, kemudian mengunjungi semua simpul di sebelah kiri dan kanan subtree. Postorder Traversal mengunjungi semua simpul di sebelah kiri dan kanan subtree terlebih dahulu, kemudian mengunjungi root node. Jobsheet ini juga menyediakan beberapa contoh kode program untuk mengimplementasikan Tree Traversal dalam bahasa pemrograman C.

# Job Sheet 13
  Jobsheet 13 ini membahas implementasi algoritma pencarian graf yaitu Depth First Search (DFS) dan Breadth First Search (BFS). Tujuan dari praktikum ini adalah agar dapat memahami konsep, operasi, dan implementasi kedua algoritma tersebut. DFS adalah metode pencarian yang mengunjungi semua simpul pada sebuah graf dengan menjelajahi sejauh mungkin sepanjang cabang sebelum berbalik (menggunakan tumpukan). Sebaliknya, BFS adalah metode pencarian yang mengunjungi semua simpul pada level yang sama sebelum beralih ke level berikutnya (menggunakan antrian).
